get_excerpt

get_excerpt(response, url_hint, body_limit=1024 * 512)

Get an excerpt from the given `response`. `url_hint` is the URL which will be used as anchor for the excerpt. The return value is a tuple in the form ``(title, body)``. If one of the two items could not be calculated it will be `None`.
